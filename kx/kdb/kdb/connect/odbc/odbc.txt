kdbsetup.exe installs the kdb odbc driver and DSN _kdb.

start a server, e.g. k db trade.t -p 2001 -d

_kdb will prompt for host:port, e.g. localhost:2001

we can add a user DSN with the odbc data source admin:

 control panel/ODBC DSN/add/KDB/ [host:port]
 the user DSN is given the name //host:port
