kdb 'insert' does 
 bulk insert on non-keyed tables
 bulk update on keyed tables
  (also insert/append new records)

for example, from a k-client:

h 4:"$update s set status=1 where s='s1'"
h 4:"$update s set status=3 where s='s3'"
h 4:"$update s set status=2 where s='s2'"
...

can be done with:

h 4:(`insert;(`s;(`s`status;(`s1`s3`s2;1 3 2))))
