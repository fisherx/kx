q merges programming(k) and database(ksql). similar to the old k. different from ksql.
added common database datatypes. 32 and 64 bit. q tables are first class.
100 times faster than ksql on the little stuff. 2 times on the big stuff.
scripts are usually half the size of the old k/ksql scripts.

datatypes
q    boolean byte short int long real float char symbol month date datetime  minute second time
ksql                    int           float      symbol month date timestamp hhmm   hhmmss time
0 date is 2000.01.01. ksql is 2035.01.01. q time is milliseconds. ksql time is float day fraction.

literals,lists,..
q			ksql
2004.12.09	date['2004-12-09']
12:34:56	time['12:34:56']
`a`bc		('a','bc')
2 34		(2,34)
(a;b)		(a,b)
([]..;		([]..,	/ create table
t[x;`f]		t[x].f	/ field access
