WARNING
 excel data is float/symbol in scalar/matrix
 working with integers, e.g. =k("!_:",4)
 working with characters, e.g. =k("`$|$:","asdf")
 for array-result: ^SHIFT ENTER 
  vector goes to one-row (for column result, e.g. =k("+,!5"))

http://kx.com/a/kd/connect/k.xls
http://kx.com/a/kd/connect/vb.txt

there is a single function k(s,...)
string s is executed and applied to the rest.

e.g. from excel97 open k.xls

a3 is =k("+",a1,a2)	' call k to add a1 and a2

other examples:
=k("\l f")	' no arguments
=k("+/",a1:a3)	' add up a1 through a3
=k("{x+y+z}",a1,a2,a3)

excel treats all values on the sheet as 64bit float or symbol.
the arrangement is either scalar(e.g. A1) or matrix(e.g. A1:A3).
k can return integers and vectors (which become float and matrices).
may want to write wrappers, e.g. fx:{[rate;n]f[rate;_ n]} / integer n

build from k.xls.
