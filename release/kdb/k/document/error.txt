reference error

lambdas({a.b}), dependencies(v..d:"a.b") and expressions(."a.b")
can have hardcoded global variable references. the system can't
reassign the parent dictionary, e.g.  a:a.b is an error

errors can be avoided by making references dynamic, e.g. {a[`b]}
