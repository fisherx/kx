load access databases into kdb ...

1. download and run kdbsetup.exe

2. find files *.mdb

3. right-click northwind.mdb and select 'kdb'

sample query:

select sum Quantity by OrderID.ShipCountry from OrderDetails

with kdb
1. click table 'OrderDetails'
2. clickdown 'select' then 'sum' then 'sum Quantity'
3. clickdown 'by' then 'OrderID' then 'OrderID.ShipCountry'
4. click 'go'
